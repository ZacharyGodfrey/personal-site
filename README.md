# personal-site

My personal website.

## Running Locally

- Run `npm i` to install dependencies
- Run `npm start` to start the local server
- Site is available at `http://localhost:8080` and will detect changes and rebuild automatically.

This site is happily hosted on [Netlify](https://www.netlify.com/).
