# Personal Website

My personal website: https://zacharygodfrey.dev

## Running Locally

- `npm i` to install dependencies
- `npm run local` to build and start the local server
- `http://localhost:8080`

The site is hosted on [Netlify](https://netlify.com) with the following settings:
